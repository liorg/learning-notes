⬅️ Backpropagation

חישוב גרדיאנט ה-loss לכל משקל — מהפלט לכניסה — על ידי כלל השרשרת.

4 שלבים
#שלבמה קורה
1Forward Passחשב פלט של כל שכבה
2Compute Lossהשווה לתשובה הנכונה
3Backward Passחשב ∂L/∂W לכל שכבה
4Update WeightsW = W − η × ∂L/∂W
W_new = W_old − η · ∂L/∂W

∂L/∂W₁ = ∂L/∂y · ∂y/∂h · ∂h/∂W₁  (כלל שרשרת)
def backward(dL_dy, x, W): dL_dW = x.T @ dL_dy # ∂L/∂W dL_db = dL_dy.sum(axis=0) # ∂L/∂b dL_dx = dL_dy @ W.T # לשכבה הקודמת return dL_dW, dL_db, dL_dx
⚠️ Vanishing Gradient: ברשתות עמוקות הגרדיאנט מתכווץ ל-0. פתרון: ReLU, BatchNorm, Skip Connections (ResNet).

רשתות לעיבוד רצפים — כל צעד מקבל קלט + hidden state מהצעד הקודם.

🔁 RNN בסיסי
hₜ = tanh(Wₕ·hₜ₋₁ + Wₓ·xₜ + b)
yₜ = Wᵧ·hₜ
⚠️ Vanishing Gradient — ברצפים ארוכים ה-RNN שוכח מידע ישן
🧠 LSTM — 3 שערים (Gates)
Gateתפקידנוסחה
Forgetמה לשכוחfₜ = σ(Wf·[hₜ₋₁,xₜ])
Inputמה להוסיףiₜ = σ(Wi·[hₜ₋₁,xₜ])
Outputמה להוציאoₜ = σ(Wo·[hₜ₋₁,xₜ])
Cₜ = fₜ ⊙ Cₜ₋₁ + iₜ ⊙ C̃ₜ
hₜ = oₜ ⊙ tanh(Cₜ)
lstm = nn.LSTM(input_size=100, hidden_size=256, num_layers=2, batch_first=True) output, (h_n, c_n) = lstm(x) # output: (batch, seq_len, 256) # h_n: hidden state אחרון | c_n: cell state אחרון

⛓️ כלל השרשרת (Chain Rule)

אם y = f(g(x)) אז:

dy/dx = (dy/dg) · (dg/dx)

dy/dx = cos(x²) · 2x

🔢 כפל מטריצות

כל שכבה ב-Neural Network היא בעצם כפל מטריצות.

📐 כלל הממדים
A(m×n) · B(n×p) = C(m×p)

Cᵢⱼ = Σₖ Aᵢₖ · Bₖⱼ
💡 מספר עמודות A חייב = מספר שורות B
A = np.array([[1,2],[3,4]]) # 2×2 B = np.array([[5,6],[7,8]]) # 2×2 C = A @ B # matrix multiply output = X @ W + b # (batch×input) @ (input×hidden) = (batch×hidden)
🔄 Transpose
(Aᵀ)ᵢⱼ = Aⱼᵢ

שימוש: חישוב גרדיאנטים ב-backprop

⚡ Hadamard ⊙
(A⊙B)ᵢⱼ = Aᵢⱼ·Bᵢⱼ

כפל איבר-איבר. שימוש: LSTM gates

📏 דמיון קוסינוס & מרחק

מדדים לדמיון בין ווקטורים — הבסיס של חיפוש סמנטי ו-embeddings.

🎯 Cosine Similarity
cos(θ) = (A·B) / (|A|·|B|)
טווח: [−1, 1]
cos = 1זהים לחלוטין
cos = 0ניצבים — לא קשורים
cos = −1מנוגדים
💡 לא תלוי באורך — רק בכיוון!
📐 מרחק אוקלידי (L2)
d(A,B) = √Σ(Aᵢ−Bᵢ)²
טווח: [0, ∞)

תלוי גם בגודל. מתאים כשהגודל משמעותי.

⚠️ High Dimensions — curse of dimensionality
def cosine_sim(a, b): return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b)) print(cosine_sim(model['king'], model['queen'])) # 0.75 print(cosine_sim(model['king'], model['apple'])) # 0.18

🎯 PCA & מטריצות מיוחדות

PCA מצמצם ממדים תוך שמירת המידע החשוב. הלב: eigenvectors ומטריצות מיוחדות.

🎯 שלבי PCA
#שלבפעולה
1CenterX = X − mean(X)
2Covariance MatrixC = XᵀX / (n−1)
3EigendecomposeC = V Λ Vᵀ
4ProjectX_new = X · V[:, :k]
🔲 מטריצה אורתוגונלית
Q·Qᵀ = I
Qᵀ = Q⁻¹

עמודות ניצבות, אורך 1. משמרת מרחקים. = eigenvectors של PCA.

📊 מטריצה אלכסונית
Λ = diag(λ₁,λ₂,...,λₙ)
Λᵢⱼ = 0 אם i≠j

רק על האלכסון. הערכים = eigenvalues = חשיבות כל כיוון.

from sklearn.decomposition import PCA pca = PCA(n_components=2) X_2d = pca.fit_transform(X) print(pca.explained_variance_ratio_.sum()) # % שנשמר print(pca.components_) # eigenvectors
🔑 SVD — Singular Value Decomposition

הכללה של PCA לכל מטריצה:

A = U · Σ · Vᵀ
Ueigenvectors של AAᵀ (left)
Σsingular values = √eigenvalues
Vᵀeigenvectors של AᵀA (right)
💡 שימוש: GloVe, LSA, Recommendation Systems, Image Compression
